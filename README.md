SYSTEM OVERVIEW:
This application is a standalone Biometric and Digital ID attendance tracker. It utilizes local machine learning (LBPH) to securely verify student faces and QR codes, alongside an immutable admin audit trail. 

INSTALLATION & SETUP:
1. No installation is required. This software is completely portable.
2. Move this entire "JAVARICE FACIAL & QR Prototype" folder to your desired location (e.g., Desktop or Documents). 
   * CRITICAL: Do not drag the .exe file out of this folder. The .exe must stay in this folder alongside its database and system files to function.
   * If you want a shortcut on your Desktop, right-click "JAVARICE FACIAL & QR Prototypeexe" -> Send to -> Desktop (Create shortcut).
3. Ensure the computer has an active 1080p webcam connected.
4. Double-click "SecureAttendance.exe" to launch the system.

DATA PRIVACY & STORAGE:
- All data is stored locally within this folder. The system does not require an active internet connection to scan faces or QR codes.
- Facial biometrics are stored as mathematical models in 'trainer.yml'. No raw photographs of faces are stored.
- Database backups can be performed simply by copying the 'attendance.db' file to a secure flash drive.

TROUBLESHOOTING:
- "Unrecognized" Face: Ensure the room is well-lit and the student is facing the camera directly. If it persists, use the Admin Dashboard to select the student and click "Retake Face".
- Camera Offline: Ensure no other applications (like Zoom or MS Teams) are currently using the webcam.
